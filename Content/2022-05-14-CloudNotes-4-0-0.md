title: CloudNotes 4.0.0 released
description: Major Update
tags: CloudNotes
---

CloudNotes 4.0.0 is a major update with focus on Markdown support. Easily work with unordered, ordered, and task lists. Additionally the support for formatting source code in Markdown is much improved.

This version also adds support for additional Nextcloud Notes features such as specifying which folder to store notes in on the server and which file extension to use.

With this release ownCloud servers are no longer supported.

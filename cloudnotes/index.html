CloudNotes

Notes app for Nextcloud

CloudNotes is a notes app for iOS (iPhone, iPod touch, and iPad) and macOS. CloudNotes syncs with the Notes app of your NextCloud server (server access is required).

Notes can be added and edited and they will be automatically saved and synced.

Preparing the NextCloud server

  • Download the server Notes app directly from GitHub. Other locations may have obsolete versions.
  • Place the zip file in nextcloud/apps on the server and unzip it.
  • Log in to the server and go to the Apps tab.
  • Enable Notes.
  • Verify that you can create and edit notes using the web interface.
  • It should now be possible to log in from the CloudNotes app on your device.
